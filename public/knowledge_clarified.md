### Calculus / Limits & Continuity
- **函数极限的统一定义** (难度: High, 高频: 8)
  - 讲解：极限是微积分的基石。ε-δ 语言精确描述了"无限接近"的过程。Heine 定理（归结原则）连接了函数极限与数列极限，是判断极限不存在的有力工具（找两个趋向于 x0 的数列，其函数值趋向不同）。
  - 二级结论：若 f(x) 在 x0 处极限存在，则局部有界。
  - 二级结论：Heine 定理：若存在两个数列趋向于 x0，其函数值趋向不同，则极限不存在。
  - 二级结论：选择题常考极限定义的逻辑辨析。
  - 二级结论：利用 Heine 定理举反例证明极限不存在。
- **无穷小比阶** (难度: Medium, 高频: 10)
  - 讲解：无穷小的比阶实际上是比较两个无穷小趋向于零的速度。高阶无穷小趋向于零的速度更快。等价无穷小替换是求极限最犀利的武器，但必须注意使用条件：通常只在乘除因子中使用。
  - 二级结论：x -> 0 时：tan x - sin x ~ x^3/2
  - 二级结论：x -> 0 时：x - sin x ~ x^3/6
  - 二级结论：x -> 0 时：arcsin x - x ~ x^3/6
  - 二级结论：x -> 0 时：tan x - x ~ x^3/3
- **泰勒公式 (Taylor Formula)** (难度: High, 高频: 9)
  - 讲解：泰勒公式是将复杂函数近似为多项式的工具。求极限用佩亚诺余项，证明不等式或中值定理用拉格朗日余项。
  - 二级结论：偶函数的泰勒展开只有偶次项，奇函数只有奇次项。
  - 二级结论：泰勒展开系数 a_n = f^(n)(0) / n!
  - 二级结论：见到“高阶导数”或“函数与多项式的关系”，优先想泰勒。
  - 二级结论：避免误区：余项类型混淆
- **间断点分类** (难度: Low, 高频: 3)
  - 讲解：间断点的分类依据是左右极限的存在性。第一类间断点（左右极限均存在）包括可去间断点和跳跃间断点。第二类间断点（左右极限至少有一个不存在）常见的是无穷间断点和振荡间断点。
  - 二级结论：分段函数的分段点是考察重点。
  - 二级结论：初等函数在定义域内连续。
  - 二级结论：避免误区：左右极限存在但不相等误判为第二类
  - 二级结论：避免误区：忽略定义域
- **泰勒公式 (Taylor's Formula)** (难度: High, 高频: 10)
  - 讲解：泰勒公式是分析学最强大的工具之一，将复杂函数近似为多项式。

### Calculus / Derivatives
- **导数定义与几何意义** (难度: Medium, 高频: 6)
  - 讲解：导数的本质是变化率。几何上，它是切线的斜率。判断分段点是否可导，必须严格使用导数定义（左右导数定义）。
  - 二级结论：奇函数的导数是偶函数，偶函数的导数是奇函数（定义域对称）。
  - 二级结论：周期函数的导数仍是周期函数，周期不变。
  - 二级结论：可导的偶函数 f(x)，必有 f'(0) = 0。
  - 二级结论：导数存在的充要条件是左右导数存在且相等。

  - 二级结论：当求导阶数 n 很大时，优先考虑莱布尼茨公式。
  - 二级结论：避免误区：组合数系数弄错
  - 二级结论：避免误区：求导阶数弄错

### Calculus / Mean Value Theorems
- **罗尔定理** (难度: Medium, 高频: 6)
  - 讲解：罗尔定理是微分中值定理的基础。三个条件缺一不可：闭区间连续、开区间可导、端点值相等。
  - 二级结论：辅助函数构造是应用罗尔定理的关键。通常将方程 f'(x)=0 还原为原函数 F(x)。
  - 二级结论：避免误区：忽略区间端点连续不可导的情况
  - 二级结论：避免误区：忘记验证三个条件
- **拉格朗日中值定理** (难度: High, 高频: 10)
  - 讲解：拉格朗日中值定理是罗尔定理的推广。它建立了函数增量与导数之间的联系。
  - 二级结论：证明 f(b)-f(a) > A 形式的不等式时，首选拉格朗日。
  - 二级结论：避免误区：忽略开区间可导的前提
  - 二级结论：避免误区：构造辅助函数错误
- **Cauchy 拉格朗日中值定理对极值与不等式的应用** (难度: Medium, 高频: 4)
  - 讲解：构造两函数 F,G，应用 Cauchy 形式得到 F'(ξ)/G'(ξ)= [F(b)-F(a)]/[G(b)-G(a)]，常用来证明不等式或估计极值。
  - 二级结论：选 G 单调且不为 0，如 G(x)=x 或 ln x；证明不等式时让右端成目标式。
  - 二级结论：重点：掌握 Cauchy 拉格朗日中值定理对极值与不等式的应用 的定义与典型题。
- **罗尔定理与零点个数** (难度: Low, 高频: 3)
  - 讲解：罗尔定理可用于证明区间内至少存在一个驻点，从而估计零点个数。
  - 二级结论：先构造满足端点函数值相等；利用导数零点与波形。
  - 二级结论：重点：掌握 罗尔定理与零点个数 的定义与典型题。
- **定积分中值定理应用** (难度: Low, 高频: 2)
  - 讲解：存在 ξ 使 ∫_a^b f = f(ξ)(b-a)，配合单调性估计积分值。
  - 二级结论：若 f 单调，可用端点值界定积分。
  - 二级结论：重点：掌握 定积分中值定理应用 的定义与典型题。

### Calculus / Integrals

  - 二级结论：避免误区：忘记加常数 C

- **凑微分法 (第一类换元)** (难度: Medium, 高频: 10)
  - 讲解：凑微分法的核心是将积分形式 ∫ f(g(x)) g'(x) dx 转化为 ∫ f(u) du。
  - 二级结论：凑微分是积分计算中最常用的技巧，必须练到条件反射。
  - 二级结论：避免误区：凑微分系数错误
  - 二级结论：避免误区：变量未完全替换
- **分部积分法** (难度: Medium, 高频: 10)
  - 讲解：分部积分法旨在将难积的 ∫ u dv 转化为易积的 ∫ v du。u 的选择遵循 "反对幂三指" 原则。
  - 二级结论：分部积分可能需要多次使用。
  - 二级结论：立志积分（表格法）可以大大加快多次分部积分的速度。
  - 二级结论：避免误区：u, v 选择不当
  - 二级结论：避免误区：符号错误

  - 二级结论：画图是解题的第一步。
  - 二级结论：注意积分上下限的确定。
  - 二级结论：避免误区：旋转轴弄错
  - 二级结论：避免误区：柱壳法公式系数 2π 漏写
- **定积分物理应用** (难度: Medium, 高频: 4)
  - 讲解：物理应用的关键是“微元法”。将总量分解为微小量的累加。
  - 二级结论：注意单位换算。

  - 二级结论：避免误区：坐标系选取不当
- **定积分的定义与性质** (难度: Medium, 高频: 9)
  - 讲解：定积分是求和的极限。掌握利用对称性（奇偶、周期）简化计算是解题关键。
  - 二级结论：奇函数在对称区间积分值为0。
  - 二级结论：偶函数在对称区间积分值为一半区间的2倍。
  - 二级结论：周期函数在任一周期长度上的积分值相等。
  - 二级结论：见到无穷项和求极限，优先考虑定积分定义。
- **牛顿-莱布尼茨公式** (难度: High, 高频: 10)
  - 讲解：微积分基本定理。变上限积分求导是每年必考点，常结合洛必达法则考察。
  - 二级结论：若 f(x) 连续，则变上限积分 F(x) 可导。
  - 二级结论：若 f(x) 可导，则 F(x) 二阶可导。
  - 二级结论：变上限积分求导公式：f(upper)*upper' - f(lower)*lower'。
  - 二级结论：避免误区：变上限求导忘记链式法则
- **分部积分法** (难度: Medium, 高频: 9)
  - 讲解：分部积分的核心是降低被积函数的复杂度。选 u 原则：反对幂三指（排在前面的选 u）。
  - 二级结论：表格法：一列求导(D)，一列积分(I)，斜对角相乘，正负交替。
  - 二级结论：表格法（DI method）可快速处理多次分部积分。
  - 二级结论：避免误区：u, v 选择错误
  - 二级结论：避免误区：符号错误
- **换元积分法** (难度: High, 高频: 10)
  - 讲解：凑微分法需要对导数公式非常熟悉。三角代换处理根式最有效。
  - 二级结论：区间再现公式：∫(a,b) f(x) dx = ∫(a,b) f(a+b-x) dx。
  - 二级结论：华里士公式 (Wallis Formula) 用于计算 sin^n x 在 [0, pi/2] 的积分。
  - 二级结论：见到 sqrt(a^2-x^2) 想到 sin，见到 sqrt(x^2+a^2) 想到 tan，见到 sqrt(x^2-a^2) 想到 sec。
  - 二级结论：避免误区：换元后忘记换限（定积分）
- **分部积分与递推公式** (难度: Medium, 高频: 4)
  - 讲解：对含幂×三角/指数常用分部积分得到递推式，减少幂次。
  - 二级结论：设 I_n，分部后移项得递推；注意收敛项边界。
  - 二级结论：重点：掌握 分部积分与递推公式 的定义与典型题。

### 高等数学 / 定积分
- **定积分存在性与达布定理** (难度: Medium, 高频: 5)
  - 讲解：闭区间有界函数黎曼可积当且仅当上下达布和极限相等；有界单调、分段连续函数必可积。
  - 二级结论：有限第一类间断点的有界函数可积。
  - 二级结论：狄利克雷函数上下和相差 1，典型不可积反例。
  - 二级结论：选择题常考“有界+可数间断”是否可积。
  - 二级结论：构造上下和证明不可积是开放性问答高频套路。
- **微积分基本定理（第一、二部分）** (难度: High, 高频: 5)
  - 讲解：第一部分：若f在[a,b]可积，F(x)=∫_a^x f(t)dt，则F连续且F' = f（几乎处处）。第二部分：若F可导且F'连续，则∫_a^b F'(x)dx = F(b)-F(a)。
  - 二级结论：可积函数原函数未必处处可导，例：f(x)=|x|/x 在0不可导。
  - 二级结论：有界可积 + 绝对连续 ⇒ 可导 a.e. 且积分换导数成立。
  - 二级结论：证明题常用“定义 + 中值定理”展示第一部分。
  - 二级结论：构造 F(x)=∫_a^x f(t)dt 作为原函数求极值/单调。
- **广义积分收敛判别（比较/极限比较/分部）** (难度: Medium, 高频: 4)
  - 讲解：常用比较：0≤f≤g 且∫g 收敛 ⇒ ∫f 收敛。极限比较：f/g→L∈(0,∞) 同敛同散；交错型用狄利克雷、阿贝尔判别；分部法求收敛域如∫0^∞ x^m e^{-ax}。
  - 二级结论：p-型积分 ∫_1^∞ 1/x^p 收敛当且仅当 p>1。
  - 二级结论：瑕积分两侧都收敛才整体收敛；若一侧发散则发散。
  - 二级结论：先与标准模型 1/x^p、e^{-ax}、sinx/x 做极限比较。
  - 二级结论：交错积分优先尝试分部或拉贝判别，避免直接比较。

  - 二级结论：Gamma-Poisson：后验形状/率累加；Normal-Normal：精度加和、均值加权。

  - 二级结论：Beta-Bernoulli：预测成功概率期望 = α/(α+β)。
  - 二级结论：可信区间随先验/数据同时变化；样本越多后验越尖锐。
  - 二级结论：遇到“下一次/未来观测”优先写预测分布，而非直接代后验均值。

### Data Structures / Linear Lists
- **线性表 - 顺序表与链表** (难度: Low, 高频: 5)
  - 讲解：顺序表支持随机访问，但插入删除需要移动大量元素；链表不支持随机访问，但插入删除只需修改指针。
  - 二级结论：链表判环：快慢指针法（快指针每次走2步，慢指针每次走1步）。
  - 二级结论：寻找链表倒数第 k 个节点：双指针法（先行指针先走 k 步）。
  - 二级结论：链表逆置是高频考点。
  - 二级结论：避免误区：链表操作忘记断链保护
- **顺序表与链表性能对比** (难度: Low, 高频: 3)
  - 讲解：顺序表随机访问 O(1) 插删 O(n)；链表随机 O(n) 插删 O(1)。
  - 二级结论：考试会让你选结构或给复杂度。
  - 二级结论：重点：掌握 顺序表与链表性能对比 的定义与典型题。

### Data Structures / Trees
- **二叉树遍历** (难度: Medium, 高频: 5)
  - 讲解：二叉树遍历分为前序 (NLR)、中序 (LNR)、后序 (LRN) 和层次遍历。
  - 二级结论：n0 = n2 + 1 (度为0的节点数总是比度为2的节点数多1)。
  - 二级结论：满二叉树高度 h = log2(n+1)。
  - 二级结论：完全二叉树高度 h = floor(log2(n)) + 1。
  - 二级结论：已知前序+中序，或后序+中序，可以唯一确定二叉树。
- **线索二叉树** (难度: Medium, 高频: 5)
  - 讲解：线索二叉树利用 n+1 个空链域存放前驱和后继信息。
  - 二级结论：中序线索二叉树找前驱后继最容易。
  - 二级结论：避免误区：空指针域利用错误
  - 二级结论：避免误区：线索化递归结束条件
- **并查集与 Kruskal 最小生成树** (难度: Low, 高频: 3)
  - 讲解：Kruskal 按边权排序+并查集合并；路径压缩+按秩合并均摊几乎 O(1)。
  - 二级结论：实现时注意边排序、判环 find(u)!=find(v)；稠密图 Prim+堆可能更优。
  - 二级结论：重点：掌握 并查集与 Kruskal 最小生成树 的定义与典型题。
- **AVL 与红黑树对比** (难度: Medium, 高频: 3)
  - 讲解：AVL 更平衡查询快，插删旋转多；红黑树旋转少插删快。
  - 二级结论：记红黑性质、最长路径 ≤2×最短。
  - 二级结论：重点：掌握 AVL 与红黑树对比 的定义与典型题。

  - 二级结论：数据库索引用 B+；非叶仅存索引。

  - 二级结论：重点：掌握 线段树与懒惰标记 的定义与典型题。
- **AVL 与红黑树对比** (难度: High, 高频: 5)
  - 讲解：AVL 更严格平衡，查询快但插入/删除旋转多；红黑树相对宽松，保证最长路径不超过最短的2倍，插入删除更快。
  - 二级结论：红黑树性质：根黑、叶黑、红节点黑父、每路径黑高相等。

  - 二级结论：手写插入删除时画出旋转步骤，标明颜色。
  - 二级结论：选择哪种树要结合读写比例。
- **B 树与 B+ 树的差异** (难度: High, 高频: 4)
  - 讲解：B 树关键字和数据都在内部节点；B+ 树数据只在叶子，叶子链表便于范围查询，内部仅存索引。
  - 二级结论：B+ 树非叶节点度数 = 指针数，便于磁盘顺序访问。
  - 二级结论：B 树查找可在内部节点结束，B+ 必到叶子。
  - 二级结论：数据库索引、文件系统多用 B+ 树。
  - 二级结论：分裂/合并时注意最小度 t 的约束。

### Computer Networks / Transport Layer
- **TCP 三次握手与四次挥手** (难度: High, 高频: 5)
  - 讲解：TCP 提供可靠的面向连接服务。三次握手建立连接，四次挥手释放连接。
  - 二级结论：TIME_WAIT 状态需等待 2MSL，确保服务端收到最后的 ACK。
  - 二级结论：避免误区：序列号 seq 和确认号 ack 计算错误
  - 二级结论：避免误区：TIME_WAIT 状态作用
- **TCP 拥塞控制** (难度: High, 高频: 5)
  - 讲解：拥塞控制防止过多数据注入网络。
  - 二级结论：发送窗口 = min(接收窗口 rwnd, 拥塞窗口 cwnd)。
  - 二级结论：避免误区：慢开始门限 ssthresh 更新规则
  - 二级结论：避免误区：3个重复ACK触发快重传
- **TCP 拥塞控制速记** (难度: Medium, 高频: 4)
  - 讲解：慢开始 cwnd=1 MSS 指数涨到 ssthresh，再线性加；超时阈值减半回慢开始；快重传/快恢复 3ACK 触发，cwnd 减半进入拥塞避免。
  - 二级结论：画 cwnd 曲线；记 ssthresh 更新：超时=阈值减半，快重传=阈值减半+进入避免。
  - 二级结论：重点：掌握 TCP 拥塞控制速记 的定义与典型题。
- **TCP 三次握手与四次挥手** (难度: Low, 高频: 4)
  - 讲解：三握确认收发能力；四挥分离半关闭与 TIME_WAIT。
  - 二级结论：TIME_WAIT=2MSL；半关闭一端可收不可发。
  - 二级结论：重点：掌握 TCP 三次握手与四次挥手 的定义与典型题。
- **UDP 与 TCP 对比** (难度: Low, 高频: 3)
  - 讲解：UDP 无连接、不可靠、报文边界；TCP 可靠、字节流。
  - 二级结论：实时/流媒体用 UDP；文件/可靠用 TCP。
  - 二级结论：重点：掌握 UDP 与 TCP 对比 的定义与典型题。
- **流量控制 vs 拥塞控制** (难度: Low, 高频: 2)
  - 讲解：流控管 rwnd，拥塞管 cwnd；实际窗口=min(rwnd,cwnd)。
  - 二级结论：区分端到端/网络层面。
  - 二级结论：重点：掌握 流量控制 vs 拥塞控制 的定义与典型题。
- **选择重传 SR 与回退 N GBN** (难度: Low, 高频: 2)
  - 讲解：GBN 累计确认回退窗口；SR 独立确认需缓存乱序。
  - 二级结论：SR 缓冲/序号空间更大。
  - 二级结论：重点：掌握 选择重传 SR 与回退 N GBN 的定义与典型题。
- **TCP 三次握手与四次挥手** (难度: Medium, 高频: 5)
  - 讲解：三次握手：SYN→SYN+ACK→ACK，建立双向初始序列号。四次挥手：FIN/ACK 分离，TIME_WAIT 保证旧报文消逝。
  - 二级结论：握手三次足够防止旧连接误触发。
  - 二级结论：TIME_WAIT 持续 2MSL，避免延迟报文影响新连接。
  - 二级结论：序号与确认号计算题要明确“下一字节序号”。
  - 二级结论：服务器崩溃重启导致半开连接，可用保活探测。
- **滑动窗口与流量控制** (难度: Medium, 高频: 4)
  - 讲解：发送窗口限制未确认的数据量，接收窗口通告可用缓存，双方取最小值。超时重传基于 RTT 估计。
  - 二级结论：快速重传需 3 个冗余 ACK，配合快速恢复。
  - 二级结论：零窗口会导致死锁，TCP 用持续计时器与零窗口探测。
  - 二级结论：窗口移动示意图需标清 SND.UNA/SND.NXT。
  - 二级结论：拥塞控制与流量控制同时生效，取更小窗口。
- **TCP 拥塞控制四阶段** (难度: High, 高频: 5)
  - 讲解：慢启动指数增长到阈值；拥塞避免线性增长；快重传+快恢复遇到 3 重 ACK 将阈值减半并进入拥塞避免；超时则阈值减半并重回慢启动。
  - 二级结论：ssthresh 是分界；cwnd 不会小于 1 MSS。
  - 二级结论：Reno 与 NewReno、CUBIC 的恢复细节不同。
  - 二级结论：画 cwnd 随时间曲线；标注事件点（超时/3ACK）。
  - 二级结论：吞吐估算：吞吐≈MSS/RTT × √(3/2p)。
- **TCP 高频考点汇总（OCR 模拟卷）** (难度: Medium, 高频: 9)
  - 讲解：模拟卷出现频率最高的 TCP 知识点：三次握手/四次挥手时序，滑动窗口与吞吐率估算，拥塞控制四阶段（慢启动、拥塞避免、快重传、快恢复），TIME_WAIT 含义。
  - 二级结论：计算题先写出 cwnd 演变表；注意超时与 3 重 ACK 的分界。
  - 二级结论：时序题画出 seq/ack 轨迹，强调“第二次握手携带 ACK”。

  - 二级结论：安全状态≠无死锁，但死锁必为不安全。

- **内核线程与用户线程** (难度: Low, 高频: 3)
  - 讲解：用户级线程切换快但一次阻塞会阻塞整个进程；内核线程由内核调度，能利用多核但切换成本高。
  - 二级结论：1:1、N:1、M:N 三种映射模型；M:N 需两级调度。
  - 二级结论：混合模型能兼顾多核与轻量切换。
  - 二级结论：解释阻塞问题：系统调用在用户线程模型会阻塞整个 LWP。
  - 二级结论：性能分析写明切换发生在用户态或内核态。

### Operating Systems / File Systems
- **索引节点与块地址** (难度: Medium, 高频: 4)
  - 讲解：i-node 存权限、时间戳、指向数据块的直接/一次间接/二次间接/三次间接地址，支持大文件扩展。
  - 二级结论：软链接只保存路径，不增加 i-node 引用计数。
  - 二级结论：硬链接共享 i-node，删除需计数为0才释放。
  - 二级结论：计算最大文件大小：直接块+间接块容量求和。
  - 二级结论：别把目录项与 i-node 搞混，目录文件本身也有 i-node。
- **日志文件系统与写放大** (难度: Medium, 高频: 2)
  - 讲解：写前日志：先写日志再落盘数据，崩溃后回放保证一致性。顺序写减少寻道但会产生写放大。
  - 二级结论：元数据日记模式只记录元数据，数据写放松。
  - 二级结论：全日记模式最安全但开销最大。
  - 二级结论：解释崩溃恢复步骤：重放已提交事务、回滚未完成事务。
  - 二级结论：与 Copy-on-Write（如 ZFS）比较：后者直接写新块。
- **虚拟文件系统 VFS 层** (难度: Low, 高频: 2)
  - 讲解：VFS 抽象统一接口（超级块、inode、dentry、file），允许不同具体文件系统共存。系统调用通过 VFS 分发到具体实现。
  - 二级结论：页缓存由 VFS 统一管理，读写路径共享缓存。
  - 二级结论：挂载操作将设备与 VFS 树中的目录节点绑定。
  - 二级结论：画层次结构：应用→VFS→具体 FS→块设备。
  - 二级结论：问缓存一致性时说明页缓存与磁盘的写回策略。

### Computer Networks / Overview
- **OSI 与 TCP/IP 对照** (难度: Low, 高频: 5)
  - 讲解：OSI 七层 vs TCP/IP 四层：应用=应用+表示+会话，运输，对网际，网络接口=数据链路+物理。
  - 二级结论：实际实现以 TCP/IP 为主，但分层思想源自 OSI。
  - 二级结论：层间封装：上层数据变为下层的负载。
  - 二级结论：画分层图并标注常见协议（HTTP/TCP/IP/以太网）。
  - 二级结论：考“哪个层处理差错/流量控制”按层记忆即可。

### Computer Networks / Data Link
- **ARQ 协议对比：停等、GBN、SR** (难度: Medium, 高频: 3)
  - 讲解：停等吞吐=1/(1+2a)；GBN 在错误时回退到出错帧；SR 仅重传出错帧但需更多缓存和序号空间。
  - 二级结论：GBN 窗口≤序号空间-1；SR 窗口≤(序号空间+1)/2。
  - 二级结论：效率受信道往返时延 a 影响明显。
  - 二级结论：吞吐率计算要代入差错概率和往返时延。
  - 二级结论：序号循环问题常考 SR 窗口大小上界。

### Computer Networks / Routing
- **Dijkstra 与 Bellman-Ford 对比** (难度: Medium, 高频: 3)
  - 讲解：Dijkstra 需非负权，时间复杂度 O(E logV)；Bellman-Ford 处理负权并能检测负环，复杂度 O(VE)。
  - 二级结论：链路状态路由协议（OSPF）用 Dijkstra。
  - 二级结论：距离矢量（RIP）用 Bellman-Ford 易出现计数到无穷。
  - 二级结论：算法题：先写初始化，再迭代松弛/选最小距离节点。
  - 二级结论：检测负环：若第 V 次迭代仍可松弛则存在负环。
- **OSPF 分区与DR/BDR 选举** (难度: Medium, 高频: 2)
  - 讲解：OSPF 分为骨干区与多区域，减少 LSA 泛洪。广播网络选举 DR/BDR 降低邻接数量。
  - 二级结论：DR/BDR 选举基于优先级与 Router ID。
  - 二级结论：区域内链路状态数据库一致性靠泛洪同步。
  - 二级结论：写出 LSA 类型：1 路由、2 网络、3 汇总等。
  - 二级结论：邻接建立需 Hello 包参数匹配。

### Data Structures / Disjoint Set
- **并查集的按秩合并与路径压缩** (难度: Medium, 高频: 4)
  - 讲解：按秩/按大小合并保持树浅；路径压缩在查找时将节点直接挂到根，均摊近似 O(α(n))。
  - 二级结论：两优化同时使用效果最好。
  - 二级结论：可用于 Kruskal 算法检测成环。
  - 二级结论：写 find 时使用递归压缩路径。
  - 二级结论：不要把秩当高度，它是上界。

### Data Structures / Strings
- **KMP 与前缀函数** (难度: Medium, 高频: 4)
  - 讲解：前缀函数 π[i] 代表最长真前后缀长度。失配时跳到 π[i-1]，整体 O(n+m)。
  - 二级结论：next 数组通常右移一位，注意不同教材定义。
  - 二级结论：KMP 保证文本指针不回溯。
  - 二级结论：手推 π 表时画出前后缀对齐更快。
  - 二级结论：代码题小心数组下标，先写模板再改。
- **滚动哈希与字符串匹配** (难度: Low, 高频: 2)
  - 讲解：选择基数 B 和模 M，hash[i]= (hash[i-1]·B + s[i]) mod M。可 O(1) 比较子串哈希。
  - 二级结论：双模或 64 位自然溢出降低碰撞概率。
  - 二级结论：需预计算 B 的幂以支持区间查询。
  - 二级结论：小心整型溢出，可用 long long 或 __int128。
  - 二级结论：碰撞时仍需额外比较以确保正确。

### Data Structures / Graph
- **拓扑排序与环检测** (难度: Low, 高频: 3)
  - 讲解：入度为0队列法或 DFS 逆后序皆可。拓扑序存在 ⇔ 有向图无环。
  - 二级结论：Kahn 算法中队列弹出顺序即拓扑序。
  - 二级结论：若弹出节点数 < 总节点数，则存在环。
  - 二级结论：题目让找“可行计划顺序”多半是拓扑排序。
  - 二级结论：DFS 法要标记三色避免重复访问。
- **最小生成树：Kruskal 与 Prim** (难度: Medium, 高频: 4)
  - 讲解：Kruskal 先排序边，按权值小到大选且避免成环；Prim 从任意点扩展，使用堆 O(E logV)。
  - 二级结论：MST 适用于无向连通带权图；不唯一时可给多解。
  - 二级结论：Kruskal 依赖并查集快速判环。
  - 二级结论：手算 Kruskal 先排边列表，画并查集合并过程。
  - 二级结论：Prim 在稠密图用邻接矩阵 + O(V^2) 版本足够。
- **单源最短路：Dijkstra / Bellman-Ford / SPFA** (难度: High, 高频: 4)
  - 讲解：Dijkstra 处理非负边；Bellman-Ford 支持负权并检测负环；SPFA 是队列优化版，最坏仍 O(VE)。
  - 二级结论：若存在负环，最短路不存在；算法需报告。
  - 二级结论：多源最短路可加超级源连接所有节点。
  - 二级结论：写伪代码时标注初始化 dist=∞、源点0。
  - 二级结论：避免在含负边图上使用 Dijkstra。

### Data Structures / Heap
- **堆与优先队列操作复杂度** (难度: Low, 高频: 3)
  - 讲解：二叉堆支持插入/删顶 O(log n)，建堆 O(n)。d 叉堆降低高度但每次调整比较更多。
  - 二级结论：堆排序不稳定但原地；时间 O(n log n)。
  - 二级结论：使用向下调整/向上调整的边界要清楚。
  - 二级结论：写数组堆时左右孩子索引 2i+1/2i+2（0 基）。
  - 二级结论：大顶/小顶堆别弄反。

### Compiler / Lexical Analysis
- **正则表达式到 NFA/DFA** (难度: Medium, 高频: 3)
  - 讲解：Thompson 构造将正则变为 ε-NFA，再子集构造到 DFA，最小化 DFA 降低状态数。
  - 二级结论：ε-闭包是子集构造关键。
  - 二级结论：最小化可用分割算法（Hopcroft）。
  - 二级结论：画状态图时明确初态/终态和 ε 边。
  - 二级结论：最小化题常给等价类划分，照算法迭代即可。

### Compiler / Syntax Analysis
- **LL(1) 与 LR(1) 文法区别** (难度: High, 高频: 3)
  - 讲解：LL(1) 自顶向下预测分析需无左递归、无左因子；LR(1) 自底向上，接受更广文法，核心是项目集规范族。
  - 二级结论：FIRST/FOLLOW 决定 LL(1) 选择表。
  - 二级结论：LR(1) 项含展望符，可合并为 LALR(1) 减少状态。
  - 二级结论：消除左递归、提取左因子是构造 LL(1) 必备步骤。
  - 二级结论：LR 分析表冲突（移进/规约）要给出解决策略。

### Compiler / Intermediate Code
- **三地址码与基本块** (难度: Low, 高频: 2)
  - 讲解：三地址码形如 x = y op z。基本块入口无前驱跳入、出口无内部跳转，利于局部优化。
  - 二级结论：四元式/三元式/间接三元式是常见存储形式。
  - 二级结论：控制流图由基本块节点和转移边组成。
  - 二级结论：划分基本块：以入口、分支/跳转目标为界。
  - 二级结论：公共子表达式消除等优化在块内实现。

### Compiler / Code Generation
- **寄存器分配与图着色** (难度: Medium, 高频: 2)
  - 讲解：冲突图节点为活跃变量，边表示同时活跃。K 着色成功表示可用 K 个寄存器分配；溢出变量需访存。
  - 二级结论：活跃区间来自活跃性分析，线性扫描适合实时编译。
  - 二级结论：合并（coalescing）可减少移动指令但可能增加度数。
  - 二级结论：考试常要求画冲突图并给出着色顺序。
  - 二级结论：若无法着色，说明需要溢出并重新构图。

### Computer Networks & Systems / AI Computing Basics
- **AI 基础与硬件加速（了解）** (难度: Medium, 高频: 6)
  - 讲解：考点以“了解”为主，更多考查对并行/硬件加速的认识：矩阵乘/卷积可用 SIMD/GPU；批量与小批量梯度的差别在噪声与吞吐。
  - 二级结论：小批量梯度在收敛平滑与吞吐间折中；批量过小噪声大，过大易陷局部平坦区。
  - 二级结论：L2 正则对应权重衰减；L1 促稀疏。
  - 二级结论：GPU 对矩阵乘/卷积最有效；随机访问型算法收益有限。
  - 二级结论：回答并行/加速时强调：数据并行 + 矩阵乘加速 + 显存带宽瓶颈。

### Operating Systems / Processes & Scheduling
- **容器与虚拟化（扩展）** (难度: Medium, 高频: 6)
  - 讲解：容器 ≈ 进程 + namespace + cgroup；虚拟机有独立内核。考点常作为调度/进程管理的扩展或概念题。
  - 二级结论：常考匹配：namespace→视图隔离；cgroup→资源限制。
  - 二级结论：容器调度仍由宿主机 CFS/调度器完成。
  - 二级结论：回答差异：容器轻量、启动快但隔离弱；虚拟机隔离强但开销大。
  - 二级结论：避免误区：把 namespace 当成资源配额；忽略宿主机内核共享

  - 二级结论：重点：掌握 虚拟语气标准时态 的定义与典型题。

  - 二级结论：保持礼貌：I would appreciate it if… / I am terribly sorry for…

  - 二级结论：重点：掌握 写作衔接词速查 的定义与典型题。

  - 二级结论：常见论证套路：风险提示→条件支持→监管框架。

### Writing / Argument Essay
- **议论文全替代表** (难度: Medium, 高频: 7)

  - 二级结论：让步句：While I acknowledge that..., I still contend that...
  - 二级结论：结论句：Only by joint efforts of government and individuals can we ...
  - 二级结论：论据组合：数据 + 专家 + 个人经验（三选二）。
  - 二级结论：每段首句点题；使用 however / moreover / furthermore / in addition / in conclusion 串联。

### Ma Yuan / Political Economy
- **资本循环与周转** (难度: Medium, 高频: 5)
  - 讲解：资本循环：购买阶段（货币资本）-> 生产阶段（生产资本）-> 售卖阶段（商品资本）。
  - 二级结论：加快资本周转可以增加年剩余价值量。
  - 二级结论：重点：掌握 资本循环与周转 的定义与典型题。
- **剩余价值理论** (难度: High, 高频: 5)
  - 讲解：剩余价值是雇佣工人在剩余劳动时间内创造的被资本家无偿占有的价值。
  - 二级结论：相对剩余价值是通过缩短必要劳动时间实现的。
  - 二级结论：重点：掌握 剩余价值理论 的定义与典型题。
- **资本积累的本质与后果** (难度: Medium, 高频: 5)
  - 讲解：资本积累是资本主义发展的必然趋势。随着有机构成提高，对劳动力的需求相对减少。
  - 二级结论：资本有机构成提高是导致相对过剩人口的直接原因。
  - 二级结论：重点：掌握 资本积累的本质与后果 的定义与典型题。
- **价值规律三大表现与作用** (难度: Medium, 高频: 3)
  - 讲解：价值规律要求商品按社会必要劳动时间交换；表现为价格围绕价值上下波动、调节资源流动、促使技术进步、调节收入分配。
  - 二级结论：答题结构：原理一句→三大作用各举例（行业转移/技术革新/收入分化）。
  - 二级结论：重点：掌握 价值规律三大表现与作用 的定义与典型题。
