# 考研知识点系统性总结 & 2025 年重点预测

## 1️⃣ 总览

| 科目       | 知识点总数      | 高频（≥4 次） | 中频（2‑3 次） | 低频（≤1 次） | 难度占比（高/中/低） |
| ---------- | --------------- | ------------- | -------------- | ------------- | -------------------- |
| 数学一     | 50              | 12            | 18             | 20            | 30 % / 40 % / 30 %   |

> * 低频点 = `examFrequency ≤ 1`。
> * 难度占比基于 `difficulty` 字段的分布。

## 2️⃣ 各科目重点分析

### 📐 数学一
| 章节           | 高频知识点（示例）                                     | 关键难点                             | 推荐复习策略                                                                |
| -------------- | ------------------------------------------------------ | ------------------------------------ | --------------------------------------------------------------------------- |
| **极限与连续** | `函数极限的统一定义`、`无穷小比阶`、`极限的等价无穷小` | ε‑δ 形式的严密证明、Heine 定理的使用 | ① 练习 ε‑δ 证明模板；② 归纳法记忆等价无穷小替换表；③ 重点做历年极限综合题。 |
| **微分与积分** | `导数的定义与几何意义`、`不定积分技巧`                 | 复合函数求导、分部积分               | ① 梳理常见求导公式；② 归纳积分技巧（换元、分部、拆分）。                    |
| **线性代数**   | `矩阵的秩与逆`、`特征值分解`                           | 行列式展开、特征向量求解             | ① 多做矩阵运算题；② 熟悉常见矩阵类型（对称、正交）。                        |
| **概率统计**   | `大数定律与中心极限定理`                               | 条件概率、随机变量的分布             | ① 记忆常用分布公式；② 练习概率树与期望计算。                                |

**2025 预测重点**：极限、导数、矩阵特征值、概率大数定律。

## 5️⃣ 关键提示
1. **公式记忆**：使用项目中的 `LatexRenderer`（`FormulaBlock`）把核心公式（如 ε‑δ、主定理、TCP 三次握手时序）做成卡片，随时翻阅。
2. **频率标签**：`FrequencyBadge` 已在 UI 中加入，可在复习时快速定位高频点。
3. **错题本**：`mistakes` 字段已在数据里加入，建议在复习时将对应错题记录到个人错题本，重点突破。
4. **考纲对照**：每个科目对应的考纲章节已在 `knowledgeBase` 中标注（`category`），可在 UI 侧边栏快速筛选。

祝复习顺利，2025 考研取得佳绩！ 🚀
